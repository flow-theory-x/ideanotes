## 💡 最小実装で試せる

### Day 1 MVP
```python
# Lambda関数（最小版）
def lambda_handler(event, context):
    command = event['request']['intent']['slots']['command']['value']
    
    # EC2でClaude実行
    result = execute_on_ec2(f"claude --continue '{command}'")
    
    return {
        'response': {
            'outputSpeech': {
                'text': summarize(result)
            }
        }
    }
```
